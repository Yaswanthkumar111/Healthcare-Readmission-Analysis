# Healthcare Readmission Prediction & Analysis

## 📌 Problem Statement
Hospitals often face high readmission rates, which increase costs and indicate gaps in patient care. This project aims to analyze patient data and predict readmission risk.

---

## 🚀 Solution
Built an end-to-end data pipeline using Python, SQL, Machine Learning, and Power BI to analyze and predict patient readmission.

---

## 🛠 Tech Stack
- Python (Pandas, Scikit-learn)
- SQL (MySQL)
- Power BI

---

## 🔄 Workflow
Data Collection → Data Cleaning → Feature Engineering → SQL Analysis → Machine Learning → Dashboard

---

## 📊 Key Features
- Readmission prediction using Machine Learning  
- SQL-based analysis  
- Interactive Power BI dashboard  
- KPI tracking  

---

## 🤖 Machine Learning
Trained a Random Forest model to predict patient readmission based on:
- Age
- Billing Amount
- Length of Stay
